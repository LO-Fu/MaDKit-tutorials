Description

The MaDKit Tutorials are made of over commented source code examples, structured in packages, that describe specific topics featuring:
  • Hello World examples
  • Logging using the logging mechanism of MaDKit agents
  • Communication between agents (in progess)
  • Graphical User Interface creation in MaDKit 5
  • Simulation authoring examples (in progress)


Testing all the examples by importing MaDKit-tutorials-project.zip in Eclipse

This archive contains an Eclipse project containing source codes illustrating MaDKit with one simple class per example :
  1. Within Eclipse, do Import > Existing project > Select Archive File > ${ant.project.name}-project.zip > finish
  2. Run the example by running the main methods which are in the source files
